# Features

- Simple and easy to use validation
- Fully type-safe
- Multiple constraints per-field
- Asynchronous constraints
- Functional variant available

# Installation

With yarn:

```bash
yarn add @josephluck/valley
```

With npm:

```bash
npm i --save @josephluck/valley
```
